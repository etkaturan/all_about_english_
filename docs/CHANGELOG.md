# Changelog

All notable changes to this project, newest first.

## [0.1.0] — 2026-06-26

### Added
- Initial Next.js project scaffold (TypeScript, Tailwind v4, App Router, ESLint).
- Secure environment setup: `.env.local` (ignored) + `.env.example` (template).
- Design system: ivory/navy/royal palette + Fraunces/Work Sans fonts as tokens.
- Dark-mode toggle driven by `html.dark` class.
- Landing page sections: Nav, Hero (with glass stat cards), Lessons (interactive
  pricing tiers), BookShowcase (3D tilt), About, Footer.
- Custom paper-plane cursor with trailing dot (desktop only).
- Smooth scrolling + scroll-padding for anchor nav links.
